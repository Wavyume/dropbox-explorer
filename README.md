# Dropbox File Explorer

React веб-приложение для просмотра файлов и папок из Dropbox. Реализует базовый функционал файлового проводника с навигацией, предпросмотром файлов и интеграцией с Dropbox API.

## Функциональность

- **Навигация по папкам**: открытие папок, переход назад, возврат в корневую директорию
- **Просмотр файлов**: предпросмотр изображений, PDF, видео, текстовых файлов и документов
- **Хлебные крошки (Breadcrumbs)**: навигация по истории открытых папок
- **Панель инструментов (Toolbar)**: кнопки навигации и функциональные кнопки (часть из них - заглушки)
- **Аутентификация**: ввод и валидация Dropbox access token
- **Загрузка файлов**: возможность скачать файлы на локальное устройство

## Технологии

- **React 19** - UI библиотека
- **TypeScript** - типизация
- **Vite** - сборщик и dev-сервер
- **Dropbox SDK** - интеграция с Dropbox API
- **Lucide React** - иконки
- **CSS Modules** - стилизация компонентов

## Установка и запуск

### Требования

- Node.js 18+ 
- npm или yarn

### Шаги установки

1. Установите зависимости:
```bash
npm install
```

2. Запустите dev-сервер:
```bash
npm run dev
```

3. Откройте браузер по адресу, указанному в консоли (обычно `http://localhost:5173`)

### Сборка для production

```bash
npm run build
```

Собранные файлы будут в папке `dist/`.

Для предпросмотра production сборки:
```bash
npm run preview
```

## Получение Dropbox Access Token

1. Перейдите на [Dropbox App Console](https://www.dropbox.com/developers/apps)
2. Создайте новое приложение (или используйте существующее)
3. В настройках приложения выберите необходимые разрешения (scopes):
   - `files.content.read` - для чтения содержимого файлов
   - `files.metadata.read` - для получения метаданных файлов и папок
4. Сгенерируйте Access Token в разделе "OAuth 2" → "Generated access token"
5. Скопируйте токен и введите его в приложении при первом запуске

**Важно**: Токен хранится в localStorage браузера. Для production рекомендуется использовать более безопасные методы аутентификации (OAuth 2.0 flow).

## Структура проекта

```
src/
├── components/          # React компоненты
│   ├── Breadcrumbs/    # Навигационные хлебные крошки
│   ├── FileItem/       # Компонент файла
│   ├── FolderItem/     # Компонент папки
│   ├── PreviewModal/   # Модальное окно предпросмотра
│   ├── TokenAuth/      # Компонент аутентификации
│   └── Toolbar/        # Панель инструментов
├── features/
│   └── FileExplorer/   # Основной функционал проводника
│       ├── FileExplorer.tsx
│       └── useFileExplorer.ts  # Custom hook для логики
└── services/
    └── dropbox/        # Сервисы для работы с Dropbox API
        ├── dropboxAuth.ts      # Управление токеном
        └── dropboxService.ts   # API методы
```
